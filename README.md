# 🛒 E-Commerce Frontend

A responsive and interactive e-commerce frontend application built using HTML, CSS, and JavaScript. This project simulates core features of an online shopping platform including user authentication and product management UI.

---

## 🚀 Features

- 🔐 User Login & Registration system
- 🏠 Home page with product display
- ➕ Add Product functionality
- 🎨 Responsive UI design
- ⚡ Interactive frontend using JavaScript

---

## 🛠️ Tech Stack

- HTML5  
- CSS3  
- JavaScript
